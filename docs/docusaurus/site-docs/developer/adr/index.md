# Architecture Decision Records (ADRs)

ADRs capture important technical decisions and their rationale.

## Current ADRs

- [ADR-0001 Rule metadata contract](./0001-rule-metadata-contract.md)
- [ADR-0002 Docs sync and matrixes](./0002-docs-sync-and-matrixes.md)

## ADR conventions

Each ADR includes:

- **Context**
- **Decision**
- **Consequences**
- **Status**

When architecture changes meaningfully, add a new ADR rather than rewriting old
ones.
